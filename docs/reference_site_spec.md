## 各页结构明细

### 总览 `/`
当前快照·投研看板 → "按最新推文/新闻/披露/AI 把观点变化与风险队列重排，优先处理有边际变化的标的"。
- **今日优先队列**(核心)：先处理有边际变化的股票；每项 = 标的 + 优先级 + 看多/看空 + 高风险偏多 + 最新事件(新闻/推文/披露) + 历史信号偏正面/负面 + 边际变化描述。

### 推文 `/tweets`
最新N条公开观点 → 公司线索复核。每条推文卡：相关公司 / 时间线 / 浏览·转发·收藏 / 作者立场(看多/看空/中性)+置信度(中高..) / 估值风险 / 情绪风险 / 基本面 / 标签提取 / 公司速读 / 互动质量 / 阅读线索(优先看高收藏高浏览)。

### 股票详情 `/stocks/[ticker]`
tab：概览/观点/时间线/信号/证据。
- 核心结论：作者立场 + 价格动量(**两者必须分开处理**) + 当前归入队列(偏多但高风险) + 仓位上限提示。
- 时间线★：收盘价走势 + 首次提及 + 每次提及圆点(已做)。
- 基本面证据：营收同比/毛利率/经营/净利 + 年度数据 + 利润率趋势(营业/净利率)。
- 消息面：近N天公司新闻(只作注意力输入，不等于订单/收入)。
- 近期披露：N天X份披露，复核重点(财报/指引/现金流/风险因素/会计政策)。
- 风险：情绪风险/估值风险/拥挤风险 + 敏感度。
- 分析角度矩阵：营收趋势/利润趋势/利润率趋势。

### 提及表现 `/mentions`
"AI产业链提及后收益跟踪"大表 + 交易日对齐 + 搜索(代码/主题/观点) + 筛选(队列:全部/核心/观察/复核 · 观点:看多/中性/看空 · 收益:至今为正/为负 · 链条)。

### 战绩 `/performance`（量化回测板）
- 覆盖N股票 的胜率(跑赢SPY%) + 格子热力(窗口跑赢比例)。
- 校准曲线(置信度 vs 实际兑现)。
- 分组后续表现：分组/胜率/均值超额/中位超额/热度(超额单位百分点)。
- 按事件类型分组 / 按信息新鲜度分组(实时/T+1/周度/季度)。
- 反身性窗口：超额拆成 短期反身性/后续反应/中期信号。
- 最近N天衰减：近期胜率 vs 历史(负数=近期弱)。
- 模型分歧：LLM层与方向/视图不一致的股票。
- 隐含组合：机械等权持有，月度回报(非实盘)。

### 供应链 `/supply-chain`
- 角色分层：按供应链角色展示公司节点图谱(节点=结构关系，非观点)。
- 传导事件 + 传导提示：从高优先级事件沿供应链边传导的二阶观察清单(分数=衰减后优先级)。

### 多源 `/multi-source`
- 按多源共识给股票分层(覆盖N股票)。
- 信号源管理：已配置外部观察源，enabled 后进入共识计算(当前只有"推文"主源启用，其它为控成本暂未抓取)。
- → 我们可接：Finnhub 公司新闻作为第二源。

### 行业 `/industries`
按集群观察广度 + 行业与主题压力(提及/新闻/披露分维度)。

### 分析 `/llm`
覆盖/模型/信息变化/**边际变化** + 最新AI分析feed(每股一条：当前归入队列 + 立场 + 价格动量 + 边际消息偏正/负 + 历史信号)。≈ 我们的 /stance-changes 的展示层。

### `/sources`（关于页）
官方账号(小红书/X) + 站外导航 + 更新公告 + 用法说明(股票时间线/战绩供应链多源用法) + 反馈引导。纯静态。
